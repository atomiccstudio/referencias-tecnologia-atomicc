Como contribuir

Já pensou que massa ter um único local onde as pessoas compartilham os links de conteúdos gratuitos que lhe ajudaram? E se além da listagem desses links, fosse possível filtrar com um campo de pesquisa? Essa é a proposta desse projeto. Vamos melhorar isso juntos? Veja como contribuir!

  • Dê um fork no repositório e o clone localmente.
  • Conecte seu local com o repositório original (upstream) o adicionando como remoto através do comando git remote add upstream https://github.com/atomiccstudio/referencias-tecnologia-atomicc.git .
  • Mantenha seu repositório forkado sempre atualizado com o repositório original. Saiba como aqui.
  • Crie sua branch a partir da branch principal que se chama main (o github aboliu o uso da palavra master).
  • Execute o comando npm install ou ng serve para instalar as dependências.
  • Execute o comando npm run start:local para rodar aplicação.
  • Adicione links úteis sobre qualquer assunto de tecnologia no arquivo references.ts adicionando palavras-chave para cada link. Como nesta imagem.
  • Se quiser, você também pode contribuir desenvolvendo melhorias ou novas funcionalidades.
  • Abra um pull request para a branch main com suas modificações.

Hacktoberfest

"Todo ano, no mês de outubro, a Digital Ocean organiza um evento chamado Hacktoberfest, com intuito de fomentar e movimentar a comunidade open source, sendo também um evento para pessoas que estão iniciando a contribuição em projetos Ocean Source." Como incentivo, o Digital Ocean presenteia com 1 kit bem legal composto por uma camisa e adesivos aqueles que se cadastrarem no site e fizerem 4 pull requests no mês de outubro.

Porque contribuir na comunidade open source?

"Um projeto open source é, resumidamente, um projeto criado e mantido pelas pessoas que o utilizam. Digamos que você tenha um campo de futebol open source perto de sua casa: uma pessoa vai ajudar a cortar a grama, outra ajuda a pintar as marcações, outro ajuda a regar, mais alguém traz a bola e todos podem jogar! É um sistema colaborativo de crescimento de software. Contribuir para o open source pode ser uma maneira gratificante de aprender, ensinar e construir experiência em praticamente qualquer habilidade que você possa imaginar. Além de tudo, você pode interagir com profissionais reais do mercado e consultá-los para tirarem suas dúvidas, fazerem análise de código e darem dicas para você melhorar." (Créditos)

Atomiccas

Somos um coletivo de mulheres de TI que trabalham na Accenture, comprometido em dar empoderar e dar protagonismo as mulheres, mas não só à elas. Queremos criar oportunidades para TODOS. Apesar do coletivo se chamar AtomiccAs e ser liderado por mulheres, homens são muito bem vindos tanto para participar das nossas ações, quanto para construi-las junto com a gente. Nosso principal pilar é construção coletiva, por isso open source tem tudo a ver com a gente.

Empoderamento

O nosso propósito é encorajar, acolher e engajar pessoas para garantir o seu desenvolvimento pessoal e profissional nos pilares de carreira, bem estar e lazer.

União
